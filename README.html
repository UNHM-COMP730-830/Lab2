Lab2: Your Weight on Other Planets

Objectives

  • Review the basic object-oriented concepts: class and class instantiation.
  • Learn how to create classes and objects in Java.

Work

  1. Go to your work space.
  2. Download the Lab2 folder.

    3. $ git clone https://github.com/UNHM-COMP730-830/Lab2.git

  3. Go to Lab2_Work.
  4. Review Lab2.java, which includes main(). You don't need to make any changes to this file.
  5. Complete the Planet class in the Planet file, as specified by the following class diagram.

  6. The weight is calculated by mass * surfaceGravity.
  7. Complete the PlanetData class in the PlanetData file, as specified by the following class diagram.
